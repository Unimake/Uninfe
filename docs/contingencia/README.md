# Contingência

Esta seção reúne os procedimentos de integração do UniNFe para emissão em contingência. Escolha a modalidade permitida para o documento fiscal e para a situação operacional e, depois, siga a página correspondente.

## Modalidades documentadas

- [Contingência em formulário de segurança](formulario-de-seguranca.md)
- [Contingência SVC (RS, AN e SP)](svc.md)
- [Contingência Off-line](off-line.md)
- [Contingência EPEC](epec.md)

Em todas as modalidades, mantenha o XML original, o retorno da SEFAZ e os documentos auxiliares impressos de acordo com a legislação aplicável à UF e ao modelo fiscal.
