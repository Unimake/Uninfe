# Menu Principal

O menu principal é a primeira tela exibida ao abrir o UniNFe. Ele reúne os atalhos para configuração, suporte, validação, consulta e documentação.

## Botões

| Botão | Para que serve |
|---|---|
| Configurações | Abre a tela de configurações gerais e das empresas cadastradas. Também mostra a quantidade de empresas configuradas. Quando há senha de configuração cadastrada, o UniNFe solicita a senha antes de abrir essa tela. |
| Manual de integração | Abre o manual de integração do UniNFe no navegador. Use este botão para consultar orientações de integração entre o ERP e o UniNFe. |
| Layout | Gera e abre um PDF com informações de layout dos arquivos usados na integração. Use este botão para consultar formatos esperados pelo UniNFe. |
| Atualizar o UniNFe | Abre a tela de atualização do aplicativo. Quando há senha de configuração cadastrada, o UniNFe solicita a senha antes de abrir essa tela. |
| Situação dos serviços | Abre a consulta de disponibilidade dos serviços fiscais. Este botão aparece quando há empresa de NF-e cadastrada. |
| Cadastro de contribuinte | Abre a consulta ao cadastro de contribuinte. Este botão aparece quando há empresa de NF-e cadastrada. |
| Validar XML | Abre a ferramenta de validação manual de XML. O botão fica habilitado quando há empresas cadastradas. |
| Log | Abre a tela de logs do UniNFe, usada para consultar e excluir arquivos de log. |
| Sobre | Abre a tela com informações do aplicativo, como versão, compilação, licença e links do produto. |
| Link da Unimake | Abre o site configurado da Unimake no navegador. O link aparece quando há site configurado no aplicativo. |
