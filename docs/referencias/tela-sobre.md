# Tela Sobre

## Finalidade

A tela **Sobre** reúne informações de identificação do UniNFe e dados úteis para suporte, conferência de versão e consulta da licença de uso. Ela ajuda a confirmar qual aplicativo está em execução, qual versão está instalada, há quanto tempo o sistema está aberto e onde encontrar os links oficiais relacionados ao produto.

## Onde acessar

No menu principal, clique em **Sobre o UniNFe** ou na opção equivalente exibida com o nome do aplicativo instalado.

## Informações exibidas

| Informação | O que mostra | Como usar |
|---|---|---|
| Nome do aplicativo | Nome do produto em execução. Em compilações beta, o nome recebe a indicação de versão BETA. | Use para confirmar se está usando o aplicativo correto ao acionar suporte ou conferir uma instalação. |
| Descrição do aplicativo | Descrição curta do produto. | Ajuda a identificar a finalidade da instalação aberta. |
| Desenvolvido por | Nome da empresa responsável pelo aplicativo. | Use como referência institucional do produto. |
| Site da empresa | Link para o site configurado da empresa responsável. | Clique para abrir o site no navegador. |
| Site do produto | Link para a página do produto, quando configurado. | Clique para acessar informações específicas do UniNFe. |
| Data da última modificação | Data/hora da última modificação registrada para a aplicação. | Use em atendimento técnico para comparar instalações ou confirmar atualização. |
| Versão UniNFe | Versão do UniNFe em execução. | Informe este valor ao suporte sempre que relatar erro, comportamento diferente entre estações ou dúvida sobre atualização. |
| Aplicativo está executando há | Tempo de execução contínua da aplicação. | Ajuda a identificar se o UniNFe foi reiniciado recentemente ou está rodando há muitas horas/dias. |
| Versão .NET Framework | Versão do runtime .NET detectada no ambiente em execução. | Use em diagnóstico técnico quando houver falhas dependentes do ambiente Windows/.NET. |
| Compilação do UniNFe | Versão de compilação usada pelo aplicativo. | Use junto com a versão do UniNFe para conferência técnica da instalação. |
| Autorização de utilização | Texto da licença GNU General Public License e informações de copyright. | Consulte para entender as condições gerais de uso, redistribuição e ausência de garantia do software. |

## Botões e links

| Ação | O que acontece | Quando usar |
|---|---|---|
| Clicar no site da empresa | Abre o endereço informado no navegador padrão. Se o texto for um e-mail, abre como mensagem de e-mail. | Use para acessar o site institucional ou contato configurado. |
| Clicar no site do produto | Abre a página do produto no navegador padrão. | Use para consultar informações do UniNFe. |
| Clicar no ícone de manuais | Abre o manual do UniNFe no navegador padrão. | Use para acessar a documentação/manual oficial do produto. |

## Uso em atendimento e suporte

Ao abrir um chamado ou pedir apoio técnico, consulte esta tela e informe principalmente:

- **Versão UniNFe**.
- **Compilação do UniNFe**.
- **Versão .NET Framework**.
- **Data da última modificação**.
- **Tempo de execução contínua**.

Essas informações ajudam a comparar ambientes, confirmar se a instalação está atualizada e investigar comportamentos que podem depender de versão, compilação ou tempo de execução.

## Cuidados

- Os links são abertos pelo navegador ou cliente de e-mail padrão do Windows.
- Se o computador não tiver acesso à internet, os links externos e o manual online podem não abrir.
- A licença exibida na tela é informativa; ela não altera configurações nem executa processos fiscais.
